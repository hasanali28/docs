# Create a WhatsApp Agent

A WhatsApp Agent enables you to automate conversations with customers on WhatsApp using AI. Once connected to a WhatsApp Business number, your agent can answer questions, provide support, qualify leads, send follow-ups, and perform business workflows—all while maintaining natural conversations.

---

## Before You Begin

Before creating a WhatsApp Agent, ensure that you have:

- A Weya AI account
- A connected WhatsApp Business number
- A verified WhatsApp integration
- A Knowledge Base (recommended)
- Any required tools or integrations configured

<Tip>

Connecting a Knowledge Base before creating your agent helps provide more accurate and context-aware responses.

</Tip>

---

## Step 1: Navigate to WhatsApp Agents

1. Sign in to the Weya AI Dashboard.
2. Open the **WhatsApp Agents** section.
3. Click **Create WhatsApp Agent**.

---

## Step 2: Configure Basic Information

Provide the basic details for your agent.

| Field | Description |
|--------|-------------|
| Agent Name | The display name of your WhatsApp Agent. |
| Description | Optional description for your team. |
| Language | Primary language used for conversations. |
| WhatsApp Number | Select the connected WhatsApp Business number. |

---

## Step 3: Define Your AI Behavior

Configure how the agent should communicate with customers.

Typical configuration includes:

- System Prompt
- Welcome Message
- Conversation Style
- Personality
- Business Instructions
- Supported Languages

These settings determine how your AI responds to incoming customer messages.

---

## Step 4: Connect a Knowledge Base

Attach one or more Knowledge Bases to provide your WhatsApp Agent with business knowledge.

Knowledge Bases may include:

- Product documentation
- FAQs
- Company policies
- Training manuals
- Website content
- Support documentation

## Step 5: Configure Tools & Integrations

Your WhatsApp Agent can be extended with additional capabilities, including:

- Custom Tools
- CRM Integrations
- Webhooks
- API Integrations
- Post-conversation workflows

These integrations allow the agent to perform actions beyond answering questions.

---

## Step 6: Save the Agent

After reviewing your configuration:

1. Click **Create**.
2. Wait for the agent to be created.
3. Your WhatsApp Agent will now appear in the dashboard.

---

## Test Your Agent

Before deploying your agent to customers, send a few test messages to verify:

- Welcome messages
- AI responses
- Knowledge Base retrieval
- Tool execution
- Business workflows
- Conversation quality

Testing ensures your customers receive a smooth experience.

---

## Best Practices

- Write a detailed System Prompt that clearly defines your agent's role.
- Connect a Knowledge Base for accurate responses.
- Keep responses concise and conversational.
- Test common customer questions before deployment.
- Regularly review conversations to improve performance.
