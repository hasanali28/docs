---
title: "Get All Knowledge Bases"
description: "Fetch all Knowledge Bases for the organization."
openapi: "GET /api/knowledgeBase"
---

## Overview
Retrieve a paginated list of all Knowledge Bases belonging to the authenticated user's organization. Supports searching by keyword.

## Request
<ParamField header="Authorization" type="string" required>
Bearer token for an authenticated workspace user.
</ParamField>

<ParamField query="page" type="integer">
Page number for pagination (default is 1).
</ParamField>

<ParamField query="limit" type="integer">
Number of Knowledge Bases to fetch per page (default is 10, max is 100).
</ParamField>

<ParamField query="search" type="string">
Search keyword to filter Knowledge Bases by name, description, or status.
</ParamField>

<RequestExample>
```bash cURL
curl --location '{{baseUrl}}/api/knowledgeBase?page=1&limit=10' \
  --header 'Authorization: Bearer {{authToken}}'
```
</RequestExample>

## Error Handling

- **401 Unauthorized** – Bearer token missing, expired, or user not authenticated.
- **404 Not Found** – No Knowledge Bases found for the organization.
- **500 Internal Server Error** – Failed to fetch Knowledge Bases.
